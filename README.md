# DiceRoller App

Android-приложение Dice Roller, в котором пользователи смогут нажать кнопку в приложении, 
чтобы бросить кости. Результат броска будет показан в текстовом представлении на экране.

### Научилась
- Как добавить кнопку в приложение для Android.
- Как добавить поведение при нажатии кнопки в приложении.
- Как открыть и изменить код Activity приложения.
- Как отобразить сообщение Toast.
- Как обновить содержимое TextView во время работы приложения.
